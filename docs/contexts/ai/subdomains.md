# AI

## Baseline Subdomains

| Subdomain | Responsibility |
|---|---|
| provider-routing | 模型供應商選擇與路由 |
| model-policy | 模型能力、版本與使用政策 |
| safety-guardrail | 安全護欄與內容保護 |
| prompt-pipeline | prompt、flow、tool calling orchestration |
