## 快速入門

1. 登入帳號。
2. 建立工作區。
3. 進入 Wiki-Beta。
4. 上傳第一份文件。
5. 文件完成解析後執行 AI 問答。

## 常見操作

| 目的 | 路徑 | 條件 |
| --- | --- | --- |
| 上傳知識文件 | `/wiki-beta/documents` | 已登入 |
| RAG 問答 | `/wiki-beta/rag-query` | 至少一份 `ready` 文件 |
| 建立頁面 | `/wiki-beta/pages` | 已登入 |
| 建立結構化資料庫 | `/wiki-beta/libraries` | 已登入 |
| 管理成員 | `/organization/members` | Admin |
| 查看稽核 | `/organization/audit` | Admin |
