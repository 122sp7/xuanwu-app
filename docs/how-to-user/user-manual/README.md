## 快速入門

### 第一次使用 Xuanwu App？

1. **登入**：使用你的帳號登入。
2. **建立工作區**：點選 App Rail 左側 `+` 圖示，建立第一個工作區。
3. **進入知識庫**：點選 App Rail 中的 `BookOpen` 圖示，進入 Wiki-Beta。
4. **上傳文件**：在 Documents 頁面上傳第一份 PDF，等待解析完成。
5. **AI 問答**：文件解析完成後，前往 RAG Query 輸入問題。

詳細操作見 [user-guide.md](./user-guide.md)。
